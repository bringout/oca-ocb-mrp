# Purchase and Subcontracting Management


This bridge module adds some smart buttons between Purchase and Subcontracting
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-mrp_subcontracting_purchase
```

## Dependencies

- mrp_subcontracting
- purchase_mrp
