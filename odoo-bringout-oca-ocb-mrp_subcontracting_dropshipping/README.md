# Dropship and Subcontracting Management


This bridge module allows to manage subcontracting with the dropshipping module.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-mrp_subcontracting_dropshipping
```

## Dependencies

- mrp_subcontracting
- stock_dropshipping
