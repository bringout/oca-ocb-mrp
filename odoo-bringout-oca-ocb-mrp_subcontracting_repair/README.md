# MRP Subcontracting Repair


        Bridge module between MRP subcontracting and Repair
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-mrp_subcontracting_repair
```

## Dependencies

This addon depends on:
- mrp_subcontracting
- repair
